# WHISPER XR — Experience Spec (Freeze)

- XR is an installation (poetic, minimal), not a VR game.
- Entries:
  - /experience (exhibition mode)
  - /xr (kiosk mode)
- Modes:
  - Desktop preview (no WebXR)
  - VR mode (Quest): teleport + gaze + ray
  - Reduced mode (low quality)

- Levels (roadmap):
  - Level 1 (MVP): proximity + gaze hold 1.2–1.6s → subtle “whisper response”
  - Level 2: one micro interaction per zone
  - Level 3: guided narrative loop (beats + beacon guidance)
- Collector loop:
  - printId → /p/:id/ (share page + OG) → /prints?print=:id
  - VR shows QR to /p/:id/ (primary)
